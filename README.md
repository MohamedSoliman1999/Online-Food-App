# Online-Food-App

Video Link: https://youtu.be/bwW6Pdp3mDw

Technicals:
- RX Java.
- Room Database.
- Data Cashing.
- Retrofit.
- Online API.
- Butterknife.
- Dependency injection using Dagger2.
- Paypal.
- Firebase Authentication.
- Fresco Library.
- Okhttp Library.
- MVVM and Live data.
